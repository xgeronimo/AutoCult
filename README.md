# AutoCult

Выпускная квалификационная работа студента ФКН НИУ ВШЭ Группы БПИ223 Кузьмина Егора Владимировича

Наименование программы – «Кроссплатформенное клиент – серверное мобильное приложение “Автомобильная сервисная книжка”».
Наименование программы на английском языке – «Car Service Book Cross-Platform Client-Server Mobile Application»
Краткое наименование программы – «AutoCult».

Мобильное приложение «Автомобильная сервисная книжка» для платформ iOS и Android предназначено для удобного учета расходов и обслуживания автомобилей. Пользователь может добавить один или несколько автомобилей, указав основную информацию (марка, модель, VIN, госномер) и дополнив данные фотографиями и описанием.
